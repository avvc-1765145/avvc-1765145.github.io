Settlers of Catan Board Randomizer

We make many boards everyday

How do we randomize?

We follow 3 simple steps

We do the math

An algorithm creates multiple random board combinations where each tile, each number and each port is in a random position

You refresh to get a new board

It is as simple as it sounds like! You click a button, we show you a brand new random board, you have fun

You can customize with house rules

Don't like when 6s and 8s are together? not a problem, you can use the settings to customize the board generator to fit your needs

How did we come up with this idea?

By playing ourselves

It gets boring if every game looks the same

That is why we started placing pieces randomly. but it wasn't random enought so we started using a random number generator
